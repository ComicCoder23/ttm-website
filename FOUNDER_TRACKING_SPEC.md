# TTM Founder Partner Tracking Spec

Internal reference document for tracking all Phase 1 founding trade partner engagements.
Phase 1 is limited to **5 founding trade partners** (slots 1–5). Slot 0 is reserved for Gill (free proof pilot — see GILL_PILOT_PROOF_SPEC.md).

---

## Tracking Fields

| Field | Description |
|---|---|
| **Founder Slot** | Slot number (0 = Gill proof pilot, 1–5 = paying founder partners) |
| **Business Name** | Trading name of the business |
| **Trade** | Trade type (e.g. Plumber, Electrician, Roofer, Landscaper) |
| **Package** | Starter / Brand Launch Pack / Full Growth System |
| **Standard Price** | Published long-term rate (£297 / £597 / £997) |
| **Founder Credit** | Launch partner credit applied (£50 for slots 1–5, N/A for slot 0) |
| **Founder Price** | Price paid after credit (£247 / £547 / £947 for slots 1–5) |
| **Review Received** | Yes / No / Pending — honest Google review (only requested if client is genuinely happy) |
| **Before Screenshot** | Yes / No / Pending — screenshot of online presence at point of engagement |
| **After Screenshot** | Yes / No / Pending — screenshot post-delivery for comparison |
| **30-Day Check** | Yes / No / Pending — one-month result check-in completed |
| **Referral Asked** | Yes / No — whether referral intro was requested |
| **Referral Received** | Yes / No / Name — whether intro was made and to whom |
| **Case Study Permission** | Yes (Named) / Yes (Anonymised) / No / Pending |
| **Delivery Time** | Days from signed off to delivery complete |
| **Template Reusable** | Yes / Partial / No — whether work produced a reusable template or component |
| **Upsell Route** | Next logical service offering for this client post-delivery |

## Notes

- Standard prices (£297 / £597 / £997) remain published as the long-term investment rate on the live website.
- The £50 founder credit is a launch partner arrangement only and is not a permanent discount.
- Once all 5 founder slots (1–5) are filled, standard pricing applies and the credit closes.
- Slot 0 (Gill) is a free proof pilot and is tracked separately for portfolio and workflow documentation purposes.
- All proof assets (screenshots, testimonials, case study content) must be obtained with explicit written permission before use.
